FAST Discipline Connect

Project Description

FAST Discipline Connect is an automated platform designed to streamline the management of disciplinary actions at FAST University. The system ensures a secure and user-friendly experience for students and the disciplinary committee. Students can log in using a robust two-factor authentication (2FA) system with OTP verification, ensuring data security and confidentiality. Students can view their disciplinary cases, while the committee can effortlessly manage, update, and monitor cases on a centralized platform. Built using the MERN stack (MongoDB, Express.js, React, Node.js), this application offers a responsive, scalable, and efficient solution. The project is deployed using Fortinet, ensuring top-notch security and performance.

Key Features

  • Secure login with two-factor authentication (2FA) using OTP verification.
  • Dedicated student dashboard to view and track disciplinary cases effortlessly.
  • Streamlined case management system for the disciplinary committee to handle cases efficiently.
  • Real-time notifications to keep students and committee members updated on case progress.
  • Intuitive and user-friendly interface for smooth navigation and ease of use.
  • Scalable backend architecture powered by the MERN stack for optimal performance.
  • Robust data security with deployment on Fortinet for reliable protection.
  • Responsive design that works seamlessly across both desktop and mobile devices.

Project Duration

4 months (January 2023 - April 2023)
