News Scrapper

Project Description

News Scrapper is a web application designed to scrape and fetch the latest news articles from provided source links, ensuring users stay updated with real-time information.

Key Features

  • Seamlessly upload and process large CSV files directly into MongoDB.
  • Automatically extracts news articles and headlines from user-provided URLs using web scraping.
  • Fetches the latest news periodically to ensure data remains up-to-date.
  • Supports multiple custom sources for creating personalized news feeds.
  • Organizes output with titles, summaries, and links to articles for easy access.
  • Powered by a scalable backend with Node.js and libraries like Puppeteer or Cheerio.
  • An ideal solution for real-time news aggregation and monitoring!

Project Duration

2 months (January 2024 - February 2024)

Additional Information

This project was a collaborative effort involving a team of skilled developers, designers, and project managers. Technologies used include React.js, Node.js and MongoDB.
