NetworK Ingressed DOS

Project Description

NetworK Ingressed DOS is a simulation project that demonstrates the creation and management of a botnet, controlled through a web interface. Using technologies like React, Node.js, Python, and Socket.io, the system allows real-time management of bots to target and perform simulated attacks on various APIs. This project showcases advanced concepts in network security, automation, and real-time system interactions.

Key Features

  • Web-based interface for controlling the botnet
  • Real-time bot management using Socket.io
  • Capability to target APIs of any type
  • Simulation of botnet attacks
  • Integration of React, Node.js, and Python for seamless operation

Project Duration

1 month (November 2024)

Additional Information

This project was built using a modern tech stack including React, Node.js, Python, and Socket.io to demonstrate real-time interactions and network-level simulations.
