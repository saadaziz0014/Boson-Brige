Gazetter

Project Description

Gazetter is an interactive web app built with Next.js and Node.js, featuring a clickable world map that displays real-time climate data and top news for each country.

Key Features

  • Interactive map feature allowing users to click on a country to view its climate statistics and latest news.
  • Quick search functionality to locate any country by name.
  • Real-time updates powered by APIs like OpenWeatherMap and NewsAPI for dynamic information.
  • Fully optimized and responsive design for seamless use across desktop, tablet, and mobile devices.
  • Secure and efficient backend built with Node.js for fast data delivery.
  • A perfect platform for exploring global insights effortlessly!

Project Duration

2 months (January 2024 - June 2024)

Additional Information

This project was a collaborative effort involving a team of skilled developers, designers, and project managers. Technologies used include Next.js, Node.js, and AWS for deployment.
