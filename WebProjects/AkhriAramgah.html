Akhri Aramgah

Project Description

Akhri Aramgah is a comprehensive funeral and burial services application designed to streamline end-of-life services. Built using React, Node.js, and Firebase, the application provides a seamless platform for booking graves, transportation, catering, morgue services, and more. It also maintains detailed histories of graves and related services. As part of a larger ecosystem of mobile and web applications, Akhri Aramgah integrates all these features to offer a unified, user-friendly experience.

Key Features

  • Grave booking and management
  • Transportation and catering service bookings
  • Morgue service scheduling
  • History tracking and grave maintenance
  • Integration with a broader ecosystem of mobile and web applications
  • Real-time data handling powered by Firebase
  • Modern interface built with React and Node.js

Project Duration

3 months (August 2024 - November 2024)

Additional Information

Akhri Aramgah utilizes React and Node.js for the frontend and backend, respectively, with Firebase for real-time data handling and seamless integration across its ecosystem.
