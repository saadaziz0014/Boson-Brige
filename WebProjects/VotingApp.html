Voting App

Project Description

The Voting App is a secure and user-friendly online voting platform designed to facilitate decision-making processes for stakeholders. Built using the MERN stack (MongoDB, Express.js, React, Node.js), this application emphasizes transparency, ease of use, and robust security. It empowers users to actively participate in voting and proposal submissions with confidence and clarity.

Key Features

  • Strong two-factor authentication using email-based OTPs via Nodemailer
  • Intuitive voting interface for streamlined proposal presentations
  • User-friendly dashboard for voting proposals, history, and updates
  • Option for stakeholders to submit new ideas and proposals
  • Transparent voting process with confirmation receipts and audit logs

Project Duration

2 months (December 2024 - January 2025)

Additional Information

This project was developed using the MERN stack, showcasing expertise in web development, database management, user experience design, and security implementation.
