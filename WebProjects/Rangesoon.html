Rang-e-Soon

Project Description

Rang-e-Soon is a visually captivating front-end website designed using ReactJS. The platform highlights the natural beauty of the Soon Valley and Sakesar, aiming to promote tourism in the area. It also provides users with the functionality to send queries directly to local tour operators, making it easier to plan visits and explore the scenic attractions of the region.

Key Features

  • Showcases the scenic beauty of the Soon Valley and Sakesar
  • Interactive front-end built entirely using ReactJS
  • Contact form for sending queries to local tour operators
  • User-friendly and responsive design

Project Duration

2 weeks (January 2024)

Additional Information

Rang-e-Soon was developed as a front-end only project, focusing on promoting tourism through a visually appealing and responsive website built with ReactJS.
