Football Club Management System

Project Description

The Football Club Management System is a comprehensive platform designed to cater to the needs of a football club. With three distinct interfaces, it offers a seamless experience for fans, analysts, and admins. Fans can view stats, schedules, and team updates, analysts can manage player stats and track injuries, and admins handle player transfers, ticket sales, and financial operations. This platform simplifies football club operations through technology.

Key Features

  • Fan interface to view stats, schedules, and updates
  • Analyst interface to update player stats and manage injury records
  • Admin interface for managing player transfers and financial tasks
  • Streamlined workflows for multiple stakeholders
  • Role-based access for secure operations

Project Duration

1 month (February 2023 - March 2023)

Additional Information

This project was designed using .NET core and sql server.
