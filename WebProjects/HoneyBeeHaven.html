HoneyBee Haven

Project Description

HoneyBee Haven is a full-stack web application designed to connect stakeholders in the agriculture industry. The platform serves as a marketplace for users and businesses to buy and sell products and services while integrating advanced features such as an admin panel, payment system, and communication tools. This project aimed to streamline the agriculture supply chain through technology.

Key Features

  • Marketplace for users and businesses
  • Admin panel for platform oversight
  • Chat feature for seamless communication
  • Payment integration with EasyPaisa
  • Analytics dashboard for management insights

Project Duration

2 months (April 2023 - June 2023)

Additional Information

This project was developed using Spring Boot and Thymeleaf and MySQL as the database
