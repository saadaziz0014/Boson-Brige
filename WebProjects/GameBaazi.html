Game Baazi

Project Description

Game Baazi is the next generation of video game stores that uses modern features to cater to a decades old industry. By providing a platform where user preferences are taken into account to the integration of a chatbot and seamless admin side functionality. It is one of its kind standalone e-commerce platform for video games

Key Features

  • Fully responsive design for all devices
  • Advanced security measures and encryption
  • Chatbot integration for AI suggestions
  • Scalable architecture for future enhancements
  • Libarary System for storing game keys
  • Payment Integration using braintree
  • Analytics and charts for the admin side
  • Customer support in the form of chat
  • User preferences and history tracking

Project Duration

2 months (april 2024 - June 2024)

Additional Information

This project was done using the React and Node as the stack and SQL server as the database.
