Raston Realm

Project Description

Raston Realm is a dynamic web application revolutionizing the tourism industry. Built using .NET Core for backend operations and MySQL as the database, the platform offers seamless functionality for both clients and admins. With its elegant, user-friendly design powered by Bootstrap 5, the application ensures a smooth and efficient experience for managing tours and interactions.

Key Features

  • Dual interfaces for clients and admins
  • User-friendly and elegant design
  • Client functionalities: Sign up/in, tour slot booking, cancellation, and direct messaging
  • Admin capabilities: Tour scheduling, booking management, and responsive client interaction
  • Role-based access control via cookies for secure, tailored experiences

Project Duration

1 month (June 2023 - July 2023)

Additional Information

This project was built using .NET Core for the backend and MySQL for the database. Bootstrap 5 was used to ensure a modern and responsive design.
