Gear Grit

Project Description

Gear Grit is an adventurous cycle racing game developed in Unity, where players navigate through challenging terrains filled with various hurdles. The game features multiple levels, each with increasing difficulty, requiring players to balance speed and precision to overcome obstacles. With its smooth physics and exciting cycling adventures, Gear Grit offers a thrilling experience for racing and adventure game enthusiasts.
