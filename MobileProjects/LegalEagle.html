Legal Eagle

Project Description

Legal Eagle is a sophisticated Android application suite designed to transform the legal services landscape. The system includes three tailored applications for Users, Lawyers, and Admins, each with unique roles and functionalities. By utilizing advanced technologies like Firebase Realtime Database and external AI tools such as the Gemini API, "Legal Eagle" provides a seamless and efficient solution for connecting users with legal professionals, empowering lawyers, and ensuring smooth platform management by administrators.

Key Features

  • Browse and book legal gigs posted by qualified lawyers with advanced AI-powered search filters (e.g., Gemini API).
  • Showcase legal expertise by uploading and managing legal gigs, expanding visibility and client reach.
  • Manage user accounts, oversee transactions, and ensure efficient platform operations.
  • Built on a robust development environment for a high-quality user experience.
  • Synchronizes data across all applications in real time, ensuring consistency and accuracy.

Project Duration

2 month (April 2024 - June 2024)

Additional Information

Legal Eagle is made using Android Studio and Java while using firebase for real time database connectivity.
